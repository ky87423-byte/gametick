## 1. 한 줄 정의

**한국 게임머니 서버별 실시간 시세·차트·가격알림 플랫폼.**
gamebit.co.kr을 벤치마크하되 ① 멀티 거래소 통합 시세 ② 가격 알림 ③ programmatic SEO ④ 다국어로 차별화.

- 도메인: **gametick.co.kr** (메인, 확보 가능 확인됨 2026-06-29) + gametick.kr(방어)
- 브랜드: 겜틱 / GameTick — "주식 호가 틱(tick)" 차용, 차트·실시간 정체성
- 타겟 시장: **한국(ko) 메인 + 베트남(vi) 보조** — i18n
- 수익: **하이브리드** — 게임머니 업체 광고배너(트래픽 수익) + 자사 매입사업(lc_vn/gameboostforge) CTA 유입(전환)

## 2. gamebit 대비 차별점 (= 우리가 "더 좋게")

1. **멀티 거래소 통합** — gamebit/lc_vn은 바로템 단일 소스. 바로템 + 아이템매니아 + 아이템베이를 묶어 진짜 최저가/평균가/스프레드 제공 (핵심 우위)
2. **가격 알림** — 텔레그램/디스코드/카카오 "오렌 아데나 1,400원 이하 알림". gamebit엔 없음 → 재방문·체류 무기
3. **차트 강화** — 캔들 + 이동평균 + 거래소별 가격 비교 오버레이 (gamebit은 3분/1시간/일봉 단순 차트)
4. **programmatic SEO** — `/ko/lineage/oren` 식 서버별 랜딩 자동 생성 → "오렌 아데나 시세" 롱테일 검색 유입
5. **UX** — 다크테마·모바일 우선·빠른 로딩 (lc_vn 자산 재활용)

## 3. 기술 스택 / 아키텍처

- **스택:** Next.js + PM2 + Nginx + Certbot (lc_vn/lc_info와 동일 → 학습비용 0, 코드 재활용 극대화)
- **데이터:** 파일 기반(JSON) 또는 경량 SQLite. DB 상주 부담 회피(서버 RAM 2GB 제약).
- **i18n:** `app/[locale]/` 라우팅(ko/vi), `messages/{ko,vi}.json` 카피 관리. 기본 ko.

### ⚠️ 핵심 아키텍처 결정: 공유 수집 레이어 (반드시 지킬 것)
새 사이트가 거래소(바로템 등)를 **독립적으로 폴링하면 안 된다.**
- 이유: lc_vn이 이미 같은 VPS IP에서 바로템을 ~60초마다 긁는 중. 겜틱이 또 긁으면 동일 IP 요청량 2배 → **거래소 IP 차단 시 lc_vn까지 동반 사망.**
- 해법: **수집기 1개**(lc_vn instrumentation 확장 또는 별도 cron 워커)가 긁어서 공용 데이터 저장소(`data/*.json` 또는 SQLite)에 기록 → 겜틱은 **읽기 전용**. IP 차단 리스크 + RAM 모두 절약.
